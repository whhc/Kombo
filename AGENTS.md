## Agent skills

### Issue tracker

Issues are tracked as local markdown files under `scratch/<feature>/`. See `docs/agents/issue-tracker.md`.

### Triage labels

Five canonical status strings are used as-is (needs-triage / needs-info / ready-for-agent / ready-for-human / wontfix). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context: one `CONTEXT.md` and `docs/adr/` at the repo root. See `docs/agents/domain.md`.
